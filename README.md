**Ocular** — Browser-based RAG search engine. Tesseract OCR (server + client) over 1000+ personal files. Dual-engine search (SQLite FTS5 + IndexedDB) with sub-millisecond results. Multi-LLM cited chat (Groq LLaMA / Cerebras / Gemini fallback). Three-way Drive ingestion with watchdog auto-reindexing.

**VersoStack** — Secure task-management REST API. FastAPI + PostgreSQL (SQLModel, Alembic) + Redis caching + Docker, with automated daily email reminders.

**PawFinder** — Full-stack pet adoption platform built with a 4-person team. Java Spring Boot MVC, MySQL + Spring Data JPA, Mustache + Bootstrap front end.
